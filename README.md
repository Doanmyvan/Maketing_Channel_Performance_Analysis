# Maketing Channel Performance Analysis

## Object:
This project analyzes marketing channel performance by assessing KPIs like Cost per Acquisision (CPA), conversions, etc. as well as applies customer segmentation to compare channel impact across audience groups, offering insights to optimize targeting, strategy, and resource allocation for improved decision-making and overall campaign planning.
